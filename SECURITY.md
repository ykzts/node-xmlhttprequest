# Security Policy

## Supported Versions

We only fix vulnerabilities in the latest version.

| Version | Supported          |
| ------- | ------------------ |
| 2.1.x   | :white_check_mark: |
| < 2.0   | :x:                |

## Reporting a Vulnerability

Vulnerability reports are only accepted by email.
Emails should be sent to ykzts@desire.sh.
